## Project Description

Write a Java program to calculate the power of a number using a recursive method in Java.

Example:
```Java
        Base: 3
        Exponent: 4
        Result: 81
        3 * 3 * 3 * 3 = 81
```

Program takes base and exponent as input from a user. Program should calculate a raised to the power of b.
